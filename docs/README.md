## 기능 구현

- 비상 근무를 배정할 월과 시작 요일 입력 검증하는 기능
- 평일 비상 근무 순번대로 사원 닉네임 입력
- 휴일 비상 근무 순번대로 사원 닉네임 입력
    - 순번에는 이름이 중복되면 안된다.
    - 닉네임은 5자 이하로만 한다.
    - 근무자는 최소 5명의 근무자로 한다.
    - 35명을 초과하지 않는다.
- 평일 휴일이 변경될때 연속되면 안된다.
- 만약 연속될 경우 다음 순번을 꺼내고 연속된  순번을 이전에 넣는다.
- 목록이 완성되면 해당 일과 요일 이름을 차례대로 출력한다. 이때 휴일이라면 표시를 하여야 한다.
